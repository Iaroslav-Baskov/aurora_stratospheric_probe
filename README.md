# aurora_stratospheric_probe
## Overview
Code for esp32-based stratospheric probe. The probe will reach 30 km altitude and collect data about dist particles concentration and distribution in high atmosphere.
It collects data from much sensors, saves it to SD card and sends them by LoRa and SMS.
